# 中核概念

Takos の理解では、まず「何をどの単位で扱う platform なのか」を押さえるのが重要です。

## 概念マップ

- Workspace / Space: 境界と所有
- Repo: source と artifact の起点
- Worker / Service: deploy と routing の単位
- Resource / Binding: 状態や外部接続
- Thread / Run / Artifact: AI 実行の履歴と成果物
- Package / Ecosystem: ツール配布と統合
