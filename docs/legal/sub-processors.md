# Legal: Sub-processors Summary

> このページでわかること: sub-processor disclosure の読み方と、正本ページ。

**Template — operator review required before public launch.** This page is a
public-facing summary. The canonical published sub-processor list is
[Sub-processors](/legal/subprocessors).

## Canonical List

Use `/legal/subprocessors` as the single source of truth for provider names,
roles, purposes, data categories, region behavior, and official source links.
This summary page must not duplicate the vendor table.

## Operator Boundary

Self-hosted, on-prem, or operator-managed Takosumi Accounts planes can have a
different sub-processor set. Customers should verify the signed order form and
DPA for the actual operator and environment.

## Change Notice

New production processing by a sub-processor requires notice according to the
signed DPA. The public baseline is updated in `/legal/subprocessors`; private
vendor review, DPA, SCC, security review, and customer-specific evidence stay in
the operator evidence store.
