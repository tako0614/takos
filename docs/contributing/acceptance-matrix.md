## Remaining high-priority gaps

- Plan/Apply revalidation: long migration lock lifetime and pre-Apply
  PreparedArtifact package-resolution validation.
- Rollback/repair: rollback-with-migration semantics and RepairPlan
  rematerialization.
- Resource feature planning: required provider features, pgvector/native feature
  realization, and cross-contract rename safety.
- Canary side-effect safety: candidate-scoped network policy, queue DataContract
  policy, shadow traffic side-effect guard, and DB semantic write approval.
- Publications/security: managed projection health, ChangeSetPlan partial
  failure semantics, and native raw binding approval.
- Direct deploy image/source/bundle compilation, native env destination,
  Plan/Apply enforcement, and manifest-managed mutation protection are covered.
- Retention edge cases are now covered for rollback-window workload revisions,
  prepared-artifact TTL expiry, and provider-package retention references.
